Como fazer isso..!

Conecte o seu USB Prime no computador usando o cabo USB para atualizações de firmware. O software se conectará automaticamente ao computador caso esteja usando USB. Nota: Se você está usando a versão de iOS, Android, ou Chromebook do software, você pode precisar conectar o Hub a um computador para a primeira atualização. As próximas atualizações poderão ser disponibilizadas via Bluetooh (processo sujeito a mudanças) ver mais
